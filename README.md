# Power BI Report on Online Web Store

## 📌 Project Overview
This project is a **Business Intelligence (BI) Report** created using **Power BI** to analyze the sales performance of an online web store. The primary objective is to gain insights into sales trends, customer behavior, and product profitability to enhance business decision-making.

## 🎯 Objectives
- Develop an **interactive sales dashboard** in Power BI to visualize key performance indicators (KPIs).
- Analyze **sales trends by product sub-category**.
- Evaluate **payment mode preferences**.
- Identify **profitability by product category and customer segments**.
- Detect **monthly and quarterly sales trends**.
- Provide **actionable recommendations** for business growth.

## 🛠️ Tech Stack & Tools
- **Power BI** - For data visualization & dashboard creation.
- **Microsoft Excel / CSV** - Data source format.
- **Python / SQL (Optional)** - For data preprocessing if required.

## 📊 Key Insights & Findings
### **1️⃣ Sales Performance Analysis**
- **Top-selling categories**: Printers, Sarees, and Bookcases.
- **Highest profit category**: Printers.
- **Seasonal demand**: March sees the highest sales volume.

### **2️⃣ Payment Mode Analysis**
- **35% of total transactions are COD (Cash on Delivery)**, which increases operational costs.
- **Recommendation**: Offer **discounts for online payments** to encourage digital transactions.

### **3️⃣ Customer & Product Profitability**
- **High-value customers contribute significantly to overall profit**.
- **Recommendation**: Implement **loyalty programs** for repeat buyers.

### **4️⃣ Inventory Management Recommendations**
- Maintain **high stock levels of printers throughout the year**.
- Increase **bookcase inventory in January**.
- Adjust **clothing and electronics stock based on seasonal demand**.
